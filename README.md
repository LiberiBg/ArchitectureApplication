# Docker Configuration

## Launching the Application
To launch the application, follow these steps:
1. Ensure you have Docker and Docker Compose installed.
2. Clone the repository: `git clone https://github.com/LiberiBg/ArchitectureApplication`
3. Navigate to the project directory: `cd ArchitectureApplication`
4. Run Docker Compose: `docker-compose up --build`
5. Acess Swagger at localhost:8080 and kafka UI at localhost:8081 to see topics, consumers and producers
